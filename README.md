# AsyncStorage Example

React Native AsyncStorage kullanımını göstermek amacıyla geliştirilmiş bir örnek projedir.

Bu projede, mobil uygulamalarda sık kullanılan local storage mantığı sade ve anlaşılır bir yapı ile ele alınmıştır.

## Features
- Veri kaydetme (setItem)
- Veri okuma (getItem)
- Veri silme (removeItem)
- Basit ekran yapısı
- Temiz ve anlaşılır kod yapısı

## Use Case
- Kullanıcı bilgilerini saklama
- Tema / ayar verileri
- Basit cache işlemleri

## Tech Stack
- React Native
- JavaScript
- AsyncStorage

## Project Structure
- src/screens – AsyncStorage örnek ekranları
- src/utils – Storage yardımcı fonksiyonları
